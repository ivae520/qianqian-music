## v1.0.3 更新内容

### 新功能
- 自动更新功能 - 支持从 GitHub Release 检查并下载更新
- 更新窗口支持最小化，后台继续下载
- 启动时自动检查更新（静默模式）
- 启动时自动清理 .exe.old 残留文件

### Bug 修复
- 修复列表封面三角播放按钮交互不稳定的问题（有时出现有时不出现）
- 修复自动更新后残留 .exe.old 文件的问题
- 修复同名同歌手歌曲都显示播放高亮的问题（改用 mid 精确匹配）
- 修复部分页面切换后当前播放歌曲不高亮的问题
- 修复榜单不显示歌手信息的问题（API 字段 songer_name）
- 修复榜单页面缓存不刷新的问题
- 修复下载进度条显示在主窗口底部而非对应歌曲行的问题

### 优化
- 更新对话框美化：渐变标题栏、版本对比、Markdown 更新日志
- 托盘菜单优化：更小的字体和间距
- 音频可视化改为原地蹲跳式波动（独立随机种子 + 双频率叠加）
- 封面波动改为 3 条竖线居中显示
- 封面波动设置 WA_TransparentForMouseEvents 避免拦截鼠标事件
- 性能优化：减少线程池大小、内存缓存限制、定时器频率调整
